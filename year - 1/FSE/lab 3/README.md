# [Практическая работа 3](https://edu.grsu.by/mod/resource/view.php?id=1000161)

## B1

### Краткое описание

Программа принимает запросы о перемещении кота или мыши либо запросы о выводе текущего состоянии игры. Это продолжается до завершения игры поимкой мыши либо последним запросом. После завершения игры программа выводит расстояние пройдённое котом и мышью и была ли поймана мышь.

### Способ запуска

Для правильного запуска программы первым параметром передайте ей имя файла, содержащего входные данные. Файл должен находиться в директории, из которой вы запускаете программу.

### Формат входных данных

Первая строка содержит количество строк и столбцов соответственно, оба числа целые. Произвольное количество следующих строк содержат запросы в одном из следующие форматов

<pre>
P
M [целое число] [целое число]
C [целое число] [целое число]
</pre>
